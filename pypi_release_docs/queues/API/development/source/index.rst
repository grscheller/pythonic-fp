Pythonic FP - Queues Docuentation
=================================

PyPI project `pythonic-fp.queues <https://pypi.org/project/pythonic-fp.queues/>`_
Implementing 3 types of queue data structures.

- *class* **FIFOQueue** - First In First Out Queue
- *class* **LIFOQueue** - Last In First Out Queue
- *class* **DEQueue**   - Double Ended Queue

Part of of the
`pythonic-fp namespace projects <https://github.com/grscheller/pythonic-fp/blob/main/README.md>`_.
