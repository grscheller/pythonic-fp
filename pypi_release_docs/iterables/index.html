Pythonic FP - Functions for Iterables

Functions for iterables. Part of the PyPI pythonic-fp Namespace Projects.

Functions which
Concatenating and merging iterables
Dropping and taking values from iterables
Reducing and accumulating iterables
At all times iterator protocol is assumed to be followed
All iterators are assumed to be iterable
For all iterators `my_iter` we assume `iter(my_iter) is my_iter`
Iterables are not necessarily iterators
